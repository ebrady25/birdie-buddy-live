How to read this page. Spearman ρ measures how well BBI's predicted finish order matches the actual finish — 1.0 is perfect, 0 is random, –1.0 is anti-correlated. Brier is the average squared error between BBI's probability and the actual binary outcome — lower is better. Skill % is BBI's Brier improvement over a naive "everyone gets the same field-mean probability" baseline; positive skill means the model adds information.
